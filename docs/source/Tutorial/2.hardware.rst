连接说明
--------

ESP32S3模块与其他组件的连接方式如下：

**麦克风模块连接**

* VCC - 3.3V
* GND - GND
* WS  - GPIO4
* SCK - GPIO5
* SD  - GPIO6

**功放&扬声器模块连接**

* VCC - 3.3V
* GND - GND
* DIN - GPIO7
* BCLK - GPIO15
* LRC - GPIO16

**OLED显示模块连接**

* VCC - 3.3V
* GND - GND
* SCL - GPIO42
* SDA - GPIO41

**按钮连接**

* 音量+ - GPIO40
* 音量- - GPIO39

组装步骤
--------

1. 将ESP32S3模块插入面包板中央位置
2. 按照上述连接说明，使用跳线连接各个模块
3. 检查所有连接是否牢固，避免虚接
4. 使用USB数据线连接ESP32S3模块的USB端口
